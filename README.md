# GoodGoggles

Набор шаблонов **Brave Search Goggles** (фильтрация/переранжирование результатов) под русскоязычные сценарии: “инфо”, “фильмы”, “оценки”.

Важно: Goggles **не умеют** автоматически переключаться по словам в запросе (например, `фильм`, `оценка`) — правила применяются к *результатам* (URL/доменам), а не к тексту запроса. Для “авто-режимов” используйте отдельные Goggles + (опционально) userscript-переключатель.

Также: я не помогаю настраивать поиск под пиратские/нелицензионные источники и торренты. В шаблонах ниже — только нейтральные/легальные примеры доменов.

## Файлы

Goggles лежат в `goggles/`:

- `goggles/info.goggle` — “информация/справка” (white‑list)
- `goggles/ru-clean.goggle` — “строгий” white‑list (всё лишнее скрыто)
- `goggles/movies-legal.goggle` — “фильмы/аниме/сериалы (легально)” (white‑list)
- `goggles/download-legal.goggle` — “скачать/купить/арендовать (легально)” (white‑list)
- `goggles/ratings.goggle` — “оценки/отзывы” (white‑list)
- `goggles/no-kinopoisk.goggle` — пример точечного исключения сайта

Опционально:

- `userscript/brave-goggles-autoswitch.user.js` — userscript, который может подставлять нужный `goggles_id` по ключевым словам в запросе (нужно указать URL ваших Goggles).

## Как применить Goggle в Brave

1. Разместите `.goggle` файл по публичному URL (например, GitHub/Gist/GitLab raw‑ссылка).
2. Откройте страницу создания: `https://search.brave.com/goggles/create`
3. Вставьте URL вашего `.goggle` и нажмите Submit.
4. Затем в Brave Search откройте вкладку **Goggles** и примените ваш goggle к поиску.
